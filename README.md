# JobTap

One-man service business OS. Built with React Native + Expo + Supabase.

## Stack
- React Native + Expo
- Supabase (auth, database, storage)
- Square (payments)
- Stripe (subscriptions)
- Twilio (SMS)

## Screens (18 total)
Sprint 1: Home, Customers, Customer Detail, New Customer, New Job, Jobs/Calendar, Settings

## Setup
```bash
npm install
npx expo start
```
